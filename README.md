## Development

- **Running application in debug mode:**

    ```bash
    yarn start
    ```

- **Compiling native code:** 

    ```bash
    yarn electron:build:native
    ```

## Distribution

- Current platform:

    ```bash
    yarn dist
    ```
- Windows 32 bit:  

    ```bash
    yarn dist:windows:32
    ```
- Windows 64 bit:   

    ```bash
    yarn dist:windows:64
    ```
- Linux 32 bit:

    ```bash
    yarn dist:linux:32
    ```
- Linux 64 bit:

    ```bash
    yarn dist:linux:64
    ```
- OSX:

    ```bash
    yarn dist:osx
    ```
